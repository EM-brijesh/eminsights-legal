# Privacy Policy

_Last updated: {{25/11/2025}}_

## 1. Introduction
This Privacy Policy explains how our application (“Eminsights”) collects, uses, stores, and protects information from users while interacting with Facebook Graph API, Instagram Graph API, and related services.

By using the App, you agree to the terms described in this policy.

---

## 2. Information We Collect

### 2.1 Data Provided by Facebook & Instagram APIs  
When authorized by the user, the App may access limited data through Meta APIs, including:

- Basic Instagram account information  
- Instagram Business Account ID  
- Facebook Page details  
- Public content from Hashtag Search  
- Insights and analytics

The App **does not** collect personal login credentials or sensitive personal data.

### 2.2 Automatically Collected Data
The App may collect:

- API request logs  
- Error logs  
- Access tokens (if required to perform authorized app features)

This data is used for debugging, analytics, and improving functionality.

---

## 3. How We Use Your Information

The App uses collected data exclusively to:

- Retrieve and display social media insights  
- Fetch hashtag content  
- Analyze engagement metrics  
- Manage or moderate connected pages/accounts  
- Provide features related to social listening and analytics

We **do not** sell, share, or transfer your data to third parties.

---

## 4. Data Storage & Security

- Access tokens are securely stored.
- No passwords or sensitive user data is stored.
- All data transmitted to and from the App uses HTTPS encryption.
- Access is restricted and monitored.

If hosting on a third-party provider (e.g., AWS, Vercel, Render), we follow their security best practices as well.

---

## 5. Sharing of Data

We **do not** share or disclose user data with anyone, except:

- When required by law  
- When explicitly authorized by the user  
- When needed to fulfill the functionality of Meta APIs

No data is sold or monetized.

---

## 6. User Rights

Users may:

- Revoke permissions at any time via Facebook/Instagram settings  
- Request deletion of their data  
- Disconnect their accounts from the App  
- Request what data the App has stored about them

Once permissions are revoked, the App loses access immediately.
